# SOP_TEMPLATE_V2.md

## Propósito
Esta plantilla redefine el SOP de Auditoría IMSS como una unidad completa de procedimiento.

El objetivo es que el SOP no capture solo pasos narrativos, sino también:
- documentos,
- origen de datos,
- outputs,
- relación con otros procedimientos,
- y la interpretación estructural interna que necesita el sistema.
