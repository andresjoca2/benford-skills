# Índice de Sesiones

## Propósito
Registrar sesiones relevantes para el sistema de conocimiento de Auditoría del IMSS.

## Estructura
- `Crudo/`: sesiones, transcripts o notas todavía no destiladas
- `Procesado/`: sesiones ya resumidas o clasificadas por utilidad

## Regla
Las sesiones no son el conocimiento final. Lo importante debe destilarse después hacia:
- `01 - Pruebas IMSS`
- `02 - Documentos y Ejemplos`
- `04 - Sistema del Agente`
