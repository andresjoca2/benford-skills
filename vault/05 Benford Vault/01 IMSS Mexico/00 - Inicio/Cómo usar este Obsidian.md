# Cómo usar este Obsidian

## Propósito
Explicar de forma breve cómo navegar y trabajar dentro del Obsidian `Auditoría del IMSS`.

## Navegación básica
- Empieza en [[Home]].
- Usa [[Índice General]] para encontrar los bloques principales.
- Usa [[Mapa General del Proyecto]] para entender cómo se conecta el sistema.

## Cómo está organizado
- `01 - Pruebas IMSS` = conocimiento principal por prueba
- `02 - Documentos y Ejemplos` = biblioteca documental
- `03 - Sesiones` = registro cronológico
- `04 - Sistema del Agente` = reglas y estructura del agente
- `05 - Entregables Finales` = Excel del portal + PDF final

## Regla importante
Las notas importantes deben estar ligadas entre sí.
Si algo importante queda aislado, la navegación y el graph se degradan y hay que corregirlo.
