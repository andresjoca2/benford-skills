# Resumen del Proyecto

## Proyecto
`Auditoría del IMSS`

## Qué busca este sistema
Convertir conocimiento operativo de auditores expertos en una base estructurada, navegable y usable para trabajo humano y futura automatización.

## Ejes principales
- pruebas IMSS como unidad principal
- biblioteca documental compartida
- sesiones como input cronológico
- sistema del agente como capa de reglas
- entregables finales como capa de salida del sistema
