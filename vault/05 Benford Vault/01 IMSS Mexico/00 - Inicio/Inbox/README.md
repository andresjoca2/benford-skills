# Inbox

## Propósito
Guardar material útil todavía no clasificado o que necesita reinterpretación antes de acomodarse en la estructura final.

## Qué sí entra aquí
- borradores rescatables
- notas ambiguas
- piezas útiles sin destino claro todavía
- material puente entre V1 y la nueva estructura

## Qué no entra aquí
- sesiones con fecha clara
- documentos claramente tipificados
- contenido ya ligado a una prueba específica
- reglas del sistema
