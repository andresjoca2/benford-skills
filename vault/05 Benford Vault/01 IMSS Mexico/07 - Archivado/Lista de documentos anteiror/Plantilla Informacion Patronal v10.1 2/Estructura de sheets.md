## Nota metodológica
Estas sheets representan una estructura de salida final del sistema.
No reemplazan la organización principal por pruebas, pero sí deben usarse como referencia para mapear qué pruebas alimentan qué partes del entregable gubernamental.

## Pendiente

- prueba -> sheet
- prueba -> campo o sección
- prueba -> PDF final
- prueba -> otras pruebas
