# Documento - Acumulado de nomina

## Qué es
Archivo acumulado de nómina del cliente con detalle operativo de periodos, conceptos y variables base por trabajador.

## Para qué sirve
Se usa para construir universos de revisión y extraer importes y variables base en pruebas de nómina y prestaciones.
