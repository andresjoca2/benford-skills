# Waterfall Rules

Start with the lowest-cost source likely to pass the quality gate. Escalate only when useful count, evidence quality, contactability, or geography fit is insufficient.

Stop when:

- quality gate is met,
- next known-cost step would exceed budget,
- operator max company/person limit is reached,
- remaining sources are not configured,
- remaining candidates are duplicates, suppressed, or off-brief.

Fallback must reuse already discovered names, domains, locations, evidence, and suppression checks. Do not restart from zero.
